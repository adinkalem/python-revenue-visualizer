## 📊 About the Project

This application represents my first project built using Python.
It was developed as a practical tool for a friend studying at the Faculty of Economics, with the goal of simplifying revenue analysis and making data easier to understand through visualization.

During development, I also leveraged AI tools to accelerate learning and improve implementation, especially when working with data processing and visualization libraries.

The application is designed as a desktop dashboard that allows users to load Excel datasets and instantly gain insights into sales performance across different markets.

---

## 💡 What the Application Does

The dashboard transforms raw sales data into clear and actionable insights by:

* Aggregating revenue across multiple countries
* Visualizing market share using interactive charts
* Identifying top-performing markets
* Tracking revenue trends over multiple time periods
* Highlighting best-selling products within each market
* Detecting changes in market presence over time
* Providing simple recommendations based on data patterns

---

## 🎯 Purpose

The main purpose of this application is to support basic business and economic analysis by:

* Making complex datasets easier to interpret
* Helping users identify growth opportunities
* Supporting data-driven decision making
* Providing a simple alternative to more complex BI tools

---

As my first Python project, this application helped me understand:

* GUI development with Tkinter

* Structuring a complete real-world project

It represents a combination of learning, practical use, and problem-solving in a real scenario.
